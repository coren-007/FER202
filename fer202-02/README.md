## PersonalBudget (fer202-02)

### Install
1. Install dependencies
```
npm install
```

### Start JSON Server (port 3001)
```
npm run server
```
It serves `db.json` at http://localhost:3001

### Start React app (port 3000)
In another terminal:
```
npm start
```

### Routes
- /login
- /home

### Tech
- React 18, React Router 6, Context + useReducer, Axios, (React) Bootstrap, JSON Server

### Submit
- Zip project folder excluding `node_modules`.
